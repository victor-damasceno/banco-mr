# Descrição:

Tem a função de simular, de maneira superficial, as funcionalidades de um banco através das chamadas Activities, cujo o objetivo foi para treinar essa navegação. O app simula ações como:
- Saldo </br>
- Fatura </br>
- Transferência </br>
- Poupança </br>
